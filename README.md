# TerraEstilo

E-commerce frontend de moda casual, country e lifestyle com inspiração no Rio Grande do Sul.

## Stack

- React + Vite
- CSS puro (sem backend)

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Identidade

- Marca: **TerraEstilo**
- Paleta: verde profundo, dourado/caramelo, bege areia e marrom couro
- Funcionalidades: carrinho, favoritos, busca e filtros por categoria
